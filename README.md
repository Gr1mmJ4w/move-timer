# Move Timer

Bel sağlığı için hareket hatırlatıcı PWA.

## GitHub Pages'e deploy etme

1. GitHub'da yeni bir repository oluştur: `move-timer`
2. Bu klasördeki tüm dosyaları yükle:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/icon-192.png`
   - `icons/icon-512.png`
3. Repository ayarlarında **Pages** → **Deploy from branch: main** seç
4. Birkaç dakika sonra `https://KULLANICI_ADIN.github.io/move-timer` adresinde yayında!

## Telefona kurma

### Android (Chrome):
1. Siteyi Chrome'da aç
2. Sağ üst menü → "Ana ekrana ekle"
3. Uygulama gibi açılır

### iOS (Safari):
1. Siteyi Safari'de aç
2. Alt menüde paylaş ikonu → "Ana Ekrana Ekle"
3. Uygulama gibi açılır

## Bildirim izni
Uygulamayı açtıktan sonra "Bildirimlere izin ver" butonuna bas.
Telefon arka planda olsa bile bildirim gelir.
